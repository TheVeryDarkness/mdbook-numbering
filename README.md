# mdbook-numbering

A mdBook preprocessor that adds numbering.

- [x] Adds numbers to chapter titles
- [ ] Configurable numbering styles (e.g., "1.", "1.1.", "I.", "A.", etc.)
- [x] Adds numbers to lines in code blocks.

## Note

- Using [`highlightjs-line-numbers.js`](https://github.com/yauhenipakala/highlightjs-line-numbers.js/) to add line numbers to code blocks in the rendered HTML. The license of `highlightjs-line-numbers.js` is MIT License, and is copied to [src/highlightjs/LICENSE](./src/highlightjs/LICENSE).
